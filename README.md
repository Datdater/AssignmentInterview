# AssignmentInterview – Run with Visual Studio

## Overview

This project is a **C# Console Application** developed using **.NET**. The instructions below explain how to run the application **using Visual Studio**, which is the recommended approach for this project.

---

## Prerequisites

Make sure the following tools are installed:

* **Visual Studio 2019 or 2022**

  * Install the **.NET desktop development** workload
* **.NET SDK 6.0 or later** (usually included with Visual Studio)

---


## How to Run the Application (Visual Studio)

### Step 1: Open the Solution

1. Open **Visual Studio**
2. Click **Open a project or solution**
3. Select the file:

   ```
   StorageAndRetrieval.sln
   ```

---

### Step 2: Set Startup Project

* In **Solution Explorer**, right-click on **StorageAndRetrieval**
* Select **Set as Startup Project**

---

### Step 3: Build the Project

* From the menu, select **Build → Build Solution**
* Or use shortcut: **Ctrl + Shift + B**

---

### Step 4: Run the Application

* Press **Ctrl + F5** to run **without debugging** (recommended)
* Or press **F5** to run **with debugging**

> The console window will open and display the application output.
